Summary: Latitude vs. X

Welcome, the purpose of this project was to analyze weather conditions and changes in relation to the Earth's equator. For this analysis, data was pulled from OpenWeatherMap API to create a dataset from 500 cities

Matplotlib was leveraged once the dataset was created to analyze various weather coniditions based on the latitude of the city. Weather conditions include Temperature, Cloud, Wind, and Humidity. This website visualizes on the sources data as well as an explanation of the analysis.
